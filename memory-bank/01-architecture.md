# Architecture

## Estructura
- **Frontend**: React + Vite.
- **Routing**: react-router-dom.
- **Estilos**: Tailwind CSS.
- **Iconos**: lucide-react.

## Organización de Archivos
- `components/`: Componentes reutilizables y secciones de la Home.
- `pages/`: Páginas completas (rutas).
- Root: Configuración y Entry points.

## Patrones
- Componentes funcionales con Hooks.
- Diseño atómico simplificado (Componentes grandes por sección).
- Uso de `bg-canvas` como fondo principal y `text-ink` como color de texto principal.
