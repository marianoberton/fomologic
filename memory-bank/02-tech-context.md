# Tech Context

## Stack
- **Runtime**: Node.js
- **Framework**: React 18+
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM v6+

## Comandos
- `npm run dev`: Servidor de desarrollo.
- `npm run build`: Construcción para producción.

## Dependencias Clave
- `lucide-react`: Iconos.
- `framer-motion` (Posiblemente, para animaciones, a verificar).
